Como começar a programar

Muitos perguntam como começar a programar, qual linguagem escolher em 2014. E a primeira pergunta que sempre faço é "você sabe lógica de programação"? Sabe montar condições, loops, funções? Conhece orientação a objetos? Compilar e/ou executar um programa é trivial pra você em alguma linguagem?

Se esse passo precisa ser dado, sugiro por dois caminhos, ou pelo CodeAcademy ou pelo livro Learn to Program.

O CodeAcademy é um site onde você pode aprender o básico sobre programaçãm Python, Ruby ou Javascript. Recomendo estas e não PHP, a não ser que PHP seja o foco do seu futuro trabalho. Não que eu não goste de PHP, nada contra nenhuma linguagem, respeito a diversidade e viva a caixa de ferramentas do bom programador, porém estas 3 são mais didáticas e mais abrangentes, ou seja, você pode criar mais tipos de aplicação, desde web até aplicativos para executar no seu desktop. Talvez aquele sistema de ponto de vendas pra padaria ou lojinha do seu primo? Ou o site de casamento de um amigo? Enfim.

Outra fonte que gosto bastente é o livro Learn to Program. O livro ensina a programar utilizando Ruby como linguagem e é bastante didático e fácil de ler. Uma ótima iniciação, sem dúvida. Daí pra começar a estudar assuntos mais avançados ou codificar o sistema do seus sonhos, é só dar o próximo passo.

Se já tem base de programação, vamos ao Java. Porque Java? Porque em 2014 é ainda uma linguagem importante e, principalmente para quem está começando, fácil de ser contratado por empresas que estão desenvolvendo novos sistemas ou mantendo sistemas Java. Mas se você tem um perfil Early Adopter, vá pra... JavaScript? Scala? Clojure? Ruby, será?

Primeiramente, já instalou o Java na sua máquina?

Existem basicamente 2 versões para download, uma chamada de JRE (Jara Runtime Environment), utilizada nas máquinas que querem apenas executar um programa Java, e outra chamada de JDK (Java Development Kit), utilizada por programadores que desejam escrever, compilar e executar programas Java. Baixe e instale esta última.

Se este é o seu primeiro contato com Java você pode facilitar muito sua vida baixando a versão da JDK que já vem com o Netbeans, um dos editores de código (IDE) mais utilizados pela comunidade Java e a medida que for evoluíndo sugiro que você utilize o Eclipse. Uma versão com ambos, Netbeans e Java, pode ser obtida através do site da Oracle.

São muitos os IDEs com suporte a programação Java que o programador pode utilizar, Netbeans, Eclipse, IntelliJ Idea, entre outros. Na verdade tudo o que você precisa pra criar uma aplicação Java, seja ela um simples 'hello world' ou um grande sistema corporativo de alta complexidade, é um editor de texto e um terminal. Utilize o que achar mais adequado e produtivo.

Baixe também a documentação para consultas offline Java SE 7 Documentation
